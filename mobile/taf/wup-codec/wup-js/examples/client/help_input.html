输出流Taf.JceInputStream暴露了哪些接口？

Taf.JceInputStream用于从JCE协议数据流中读取指定的字段：
函数 描述
构造函数 开发者调用 var is = new Taf.JceInputStram(buffer)，声明一个输入流对象。

输入参数类型：
字段数据值: buffer ArrayBuffer
readBoolean(tag, require, def) 从数据流中读一个Boolean类型的字段。

输入参数类型：
字段标签值：tag byte
是否  必读：require Boolean
字段默认值：def Boolean

返回数据类型：Boolean

注意：
A1 当require为true可忽略该字段；
A2 当require为false的时候，如果找不到对应Tag的字段，该函数将返回def作为默认值；
A3 该函数在特定情况下会抛出异常，请注意捕捉。

返回数据类型：Struct

注意：
A1 函数调用方法。假设有个结构体名称为TestJCE.UserInfo。
   该函数调用方法：is.readStruct(0, true, new TestJCE.UserInfo())；
A2 当require为true的时候，使用new出来结构体中的readFrom函数从数据流中读取字段；
A3 当require为false的时候，如果找不到对应Tag的字段，该函数将返回def作为默认值；
