输出流Taf.JceOutputStream暴露了哪些接口？

Taf.JceOutputStream用于组织JCE请求：
函数 描述
构造函数 开发者调用 var os = new Taf.JceOutputStram()，声明一个输出流对象。

输入参数类型：无
getBinBuffer() 开发者调用Taf.JceoutputStram其他write函数，写入字段之后，使用该函数获取协议数据流。

返回数据类型：Taf.BinBuffer
getBuffer() 开发者调用Taf.JceoutputStram其他write函数，写入字段之后，使用该函数获取协议数据流。

返回数据类型：ArrayBuffer
writeBoolean(tag, value) 向数据流中写一个Boolean类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Boolean

返回数据类型：void
writeInt8(tag, value) 向数据流中写一个Int8类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Int8

返回数据类型：void
writeInt16(tag, value) 向数据流中写一个Int16类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Int16

返回数据类型：void
writeInt32(tag, value) 向数据流中写一个Int32类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Int32

返回数据类型：void
writeInt64(tag, value) 向数据流中写一个Int64类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Int64

返回数据类型：void
writeUInt8(tag, value) 向数据流中写一个UInt8类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value UInt8

返回数据类型：void
writeUInt16(tag, value) 向数据流中写一个UInt16类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value UInt16

返回数据类型：void
writeUInt32(tag, value) 向数据流中写一个UInt32类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value UInt32

返回数据类型：void
writeString(tag, value) 向数据流中写一个String类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value String

返回数据类型：void
writeStruct(tag, value) 向数据流中写一个自定义结构体的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Struct

返回数据类型：void
writeBytes(tag, value) 向数据流中写一个Taf.BinBuffer（vector<byte>）的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Taf.BinBuffer

返回数据类型：void
writeVector(tag, value) 向数据流中写一个Taf.Vector<T>（T不可为byte）类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Taf.Vector<T>

返回数据类型：void
writeMap(tag, value) 向数据流中写一个Taf.Map<KT, VT>类型的字段。

输入参数类型：
字段标签值: tag byte
字段数据值：value Taf.Map<KT, VT>

返回数据类型：void
